### 🚗 [Real-Time Drivable Space Segmentation](https://github.com/ShhlokRastogi/Real-Time-Drivable-Space-Segmentation)
> Computer vision system for autonomous driving — built with friends

- **~78 FPS** real-time inference on unseen dashcam footage
- **90.17% mIoU** on unseen test data, 92% on validation
- Encoder: **EfficientNet-B2** · Decoder: **DeepLabV3+ (ASPP)**
- Trained on **NuScenes + India Driving Dataset** for cross-domain generalization
- Plug-and-play inference on video streams and image folders

**Stack:** PyTorch · EfficientNet · DeepLabV3+ · Dice Loss · Focal Loss · OpenCV
